PRC Approval Path

This page shows where an FTE request is in the approval process and what the manager should do next.

  1. Approval Process Steps shows the full process from Internal FTE Committee through PRC and any appeal.
  2. Path Builder asks a few questions to find where this specific request is.
  3. Path So Far shows the route this request has taken and highlights the current point.
  4. Current Request gives the direct manager update: where it is, who has it, what to do, and what confirms it moved.

Approval Process Steps

Start with Internal FTE Committee. Approved requests move to Workday/Beeline, then PRC. Appeal only applies if PRC denies the request.
