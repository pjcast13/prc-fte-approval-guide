Radiology FTE requisitions

PRC Approval Path

This guide helps a manager figure out where an FTE request is in the approval process. It does not look up a requisition number automatically. Use what you know about the request and answer the Path Builder questions. Approval Process Steps will highlight where the request is in the full process. Current Request will show who has the next move, what the manager should do next, and what confirmation to look for before moving on.
